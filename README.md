# Dif-elmos-GhaemiV2

این مخزن شامل یک وب‌اپلیکیشن استاتیک برای آموزش و حل تعاملی معادلات دیفرانسیل مرتبه اول است. رابط کاربری کاملاً فارسی، راست‌چین و واکنش‌گراست و امکانات زیر را ارائه می‌دهد:

- تشخیص معادلات مرتبه اول در سه حالت «انتگرال مستقیم»، «جداشدنی» و «خطی»
- نمایش راهنمای حل، حل مرحله‌به‌مرحله و نمایش کامل راه‌حل با توضیح متنی فارسی
- صفحه‌کلید مجازی برای تایپ سریع نمادهای پرکاربرد معادلات دیفرانسیل
- رسم نمودار تقریبی پاسخ با روش رانگه-کوتا پس از تکمیل حل
- پشتیبانی از تم روشن و تاریک با ذخیره‌سازی ترجیح کاربر
- آماده برای استقرار مستقیم روی Vercel با استفاده از `vercel.json`

## نحوه اجرا در محیط توسعه

1. وابستگی خاصی نیاز نیست؛ پروژه یک وب‌اپلیکیشن استاتیک است.
2. برای اجرای محلی می‌توانید از یک سرور ساده استفاده کنید:

```bash
npm install --global serve
serve .
```

سپس مرورگر را باز کرده و به آدرس اعلام شده مراجعه کنید.

## استقرار روی Vercel

- کافی است مخزن را در Vercel وارد یا Deploy کنید.
- فایل `vercel.json` پروژه را به عنوان اپلیکیشن استاتیک پیکربندی می‌کند.
- خروجی نهایی در همان ریشه پروژه (`index.html`) قرار دارد و نیازی به فرمان build نیست.

## محدودیت‌های حل‌کننده

حل‌کننده داخلی برای معادلات مرتبه اول طراحی شده است و از انتگرال‌گیری نمادین پایه برای توابع متداول (چندجمله‌ای‌ها، توابع نمایی و مثلثاتی رایج) استفاده می‌کند. برای معادلات پیچیده‌تر پیام راهنما نمایش داده می‌شود تا کاربر بداند که معادله در این نسخه پشتیبانی نمی‌شود.
